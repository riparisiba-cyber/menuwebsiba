🔥 Firebase — Sync automatico + Registrazione clienti

Con Firebase configurato tutti i dispositivi ricevono il menu aggiornato e inviano ordini automaticamente, senza inserire nulla.
Vai su console.firebase.google.com → il tuo progetto → ⚙️ Impostazioni → Le tue app → sezione Configurazione SDK.

Come trovare il Database URL:
Firebase Console → Realtime Database → crea il database → copia l'URL tipo https://progetto-rtdb.firebaseio.com
Regole database (incolla nel tab Regole):
{ "rules": { ".read": true, ".write": true } }
📲 Sincronizzazione permanente su tutti i dispositivi
Per far funzionare il menu su qualsiasi dispositivo senza link speciale, incolla la config Firebase direttamente nel file menu-app-leggimenu.html.
1. Clicca "📋 Copia config" qui sotto  ·  2. Apri il file HTML  ·  3. Trova la riga const FIREBASE_CFG_EMBEDDED = null;  ·  4. Sostituiscila con il testo copiato  ·  5. Carica il file su GitHub
⚡ Authentication: Firebase Console → Authentication → Sign-in method → abilita Email/Password.
🌐 Domini: Authentication → Settings → Domini autorizzati → aggiungi il tuo dominio.

📧 Email Notifiche (EmailJS)

Quando un ordine passa a Pronto ✅, il sistema invia automaticamente una email al cliente.
Registrati gratis su emailjs.com → crea un Service (Gmail, Outlook…) e un Template, poi inserisci le credenziali qui.

Variabili del template EmailJS:
{{to_name}} · {{to_email}} · {{order_id}} · {{restaurant_name}} · {{items_list}} · {{order_total}} · {{table_info}}
Esempio oggetto: "Il tuo ordine {{order_id}} è pronto! 🎉" — Assicurati di impostare To Email = {{to_email}} nel template.

🔑 Sync Cloud (JSONBin.io)

Registrati gratis su jsonbin.io → vai in API Keys → copia la Secret Key e incollala qui.

Hai già un Bin ID? Incollalo e clicca Imposta. Oppure salva le impostazioni per crearne uno automaticamente.

Condividi questo link con i clienti — si apre il menu già connesso, su qualsiasi dispositivo, senza inserire nulla.

⚠️ Il link contiene la API Key — usalo solo su connessioni sicure (QR code, WhatsApp privato, ecc.)
